# WatchTogether Frontend

Static frontend for GitHub Pages.

## Use

Open the site with a backend URL:

```text
?api=https://your-backend.example.com/
```

The frontend stores the backend URL in local storage after the first use.

## Files

- `index.html`
- `src/app.js`
- `extension/`

## Notes

This directory is static and can be deployed to GitHub Pages.
