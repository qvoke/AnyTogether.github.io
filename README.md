# WatchTogether Frontend

Static frontend for GitHub Pages.

## Use

The frontend uses `https://anytogether-backend.onrender.com/` by default.
You can override it with `?api=https://your-backend.example.com/` if needed.

## Files

- `index.html`
- `src/app.js`
- `extension/`

## Notes

This directory is static and can be deployed to GitHub Pages.
